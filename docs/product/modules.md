# Syner Cortex - Product Modules & Personas

## 1. SaaS Modules Registry
The platform is designed around 8 modules that can be dynamically enabled/disabled per tenant:
- **Cortex Vault:** Drag-and-drop secure document parsing (PDF, TXT, MD) and RAG vector ingestion.
- **Cortex Chat:** Conversational agent retrieving indexed knowledge with expandable citations.
- **Cortex Diagnose:** 360-degree company questionnaires generating SWOT matrices and recommendations.
- **Cortex Roadmap:** Action items backlogs organized in 30/60/90-day execution horizons.
- **Cortex Reports:** Preview and export compiled boardroom executive briefs.
- **Cortex Agents (Future):** Specialized consulting domain agents (Legal, PM, Finance, etc.).
- **Cortex Insights (Future):** Automated KPI metric checks and alert triggers.
- **Cortex Boardroom (Future):** Dynamic interactive charts dashboard for C-levels.

## 2. User Personas
- **Founders:** Fast feedback, SWOT checks, investor pitch roadmaps.
- **SMEs:** Process mapping, operational bottlenecks detection, role guides.
- **Corporate Execs:** Information safety, audit log checks, printable summaries.
- **Syner Consultants:** Customer data ingestion, scaling advisory methods.

## 3. Commercial Tiers
1. **Starter:** Single workspace, basic RAG chat, business diagnosis SWOT, executive brief.
2. **Business:** Multi-workspaces, 30/60/90 roadmap tracking, user role management.
3. **Enterprise:** Custom agents, advanced activity audit logging, private VPC deployment.
